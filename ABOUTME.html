为什么要写博客

技术上自学 重度依赖 google && github

有很多想法，也解决过很多问题，都没有很好的记录下来

从知道 Blog 这个名词开始想写博客到现在已经有十三年了

直到现在才开始做这件事情，实在很惭愧。原因如下：

  • 目前的工作生活状态才能安静下来写点东西
  • 技术上觉得可以拿些值得记录的东西出来
  • 目前的工具才基本满足了我对博客工具的要求
    1. 使用 类似 wiki 的轻量标记语言书写文章
    2. 所以也不需要后台管理功能
    3. 能使用 vim 编辑文章
    4. 能很方便的发布 更新
